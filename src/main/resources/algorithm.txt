Описание алгоритма сходства инициалов.
На вход поступает 2 строки для сравнения, пример:
Чеботарёв, Николай Григорьевич
Н. Г. Чеботарев

1) У строк удаляются разделители, замена ё на е, преобразование в массивы
Чеботарев Николай Григорьевич
Н Г Чеботарев

2) У массивов удаляются общие элементы
Николай Григорьевич
Н Г

3) По каждому элементу сравнивается первый символ и возвращается результат
true
